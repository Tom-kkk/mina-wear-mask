
# mina-wear-mask
A mini-program on WeChat platform that enhances user experience by decorating avatar.
Accumulated over 100k users.

充分利用的流量运营思维包括取名截流、公众号文章引流、广告放置策略优化，累计获得10w+用户。
核心代码开发时间为两个周末。
